# Data Engineer

## Architecture Highlights
- **Data Lake Layers**: Integrated **Amazon S3** for Bronze/Silver layers and **Amazon Redshift Serverless** for Gold-layer analytics.
- **Serverless Orchestration**: Designed **serverless data pipelines** using **AWS Glue, AWS Lambda, SQS, and SNS** for near real-time data ingestion and transformation.
- **Security & Governance**: Enforced **S3 lifecycle policies, KMS encryption, and CloudWatch monitoring** for optimized security and cost governance.
- **AI/ML Integration**: Leveraged **Amazon SageMaker** for predictive modeling and **AWS Glue Data Catalog** for centralized metadata management.
